What's inside

Three tools.
One simple URL.

Everything you need for quick developer sharing. Nothing you don't.

01
Most used
URL Shortener

Paste a long URL, get a clean short link. Add a custom alias, track clicks, set an expiry date. That's the whole feature set — intentionally simple, fast, and reliable.

custom-alias click-tracking qr-code expiry
02
File Sharing

Drag a file in, get a direct download link instantly. Files auto-expire so you're never hoarding junk. Supports preview for images and text files.

drag-drop auto-expiry direct-dl preview
03
Code Paste

Share code snippets with proper syntax highlighting. Burn-after-read mode for sensitive content. Fifteen+ languages supported right out of the box.

syntax-hl burn-after-read 15+ langs shareable-link
snippet.ts
const shorten = async (url: string) => {
   // POST to /api/shorten
   const res = await fetch(`/api`, {
     method: 'POST',
     body: JSON.stringify({ url })
  });
   return res.json();
};
